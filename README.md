## Homework task 5
## Using
 ```bash
# Install dependencies
$ npm install

# Run the app in the development mode and open page
$ npm run start

# Build the app for production
$ npm run build

# Run linter
$ npm run eslint

# Create commit
$ npm run commit

# Run tests
$ npm run test

# Deploy the project on gh-pages
$ npm run test
 ```
